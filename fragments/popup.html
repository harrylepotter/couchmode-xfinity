CouchMode

Couchsit for Xfinity Live TV.

Getting started

Couchsit is a Chrome Extension that turns your browser into a set-top box for watching live television.
A minimalist onscreen display lets you see the channel you"re watching

It is somewhat similar to Vimeo Couch mode or Youtube leanback mode, but for live TV.

To get started, you"ll need:

  1. A Comcast Xfinity account with subscription to channels. "remember me" must be used to store the account details in the browser
  2. A remote and remote software - For mac, I recommend Remote Buddy
  3. Map your remote control buttons to the following:
    • o Channel Up
    • p Channel Down
    • w Change channel to selection

That"s about all there is to it.
